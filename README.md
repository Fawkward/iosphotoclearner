# PhotoSwipe

Минимальное iOS-приложение для удаления фоток свайпами (как Tinder).
- Свайп влево → пометить в корзину (мгновенно, без диалогов)
- Свайп вправо → оставить (пропустить)
- Кнопка "🗑 N" сверху → удалить всю пачку одним системным подтверждением
- Кнопка ↩ сверху → отменить последнее действие

Собрано так, чтобы можно было компилировать в GitHub Actions без мака и ставить на айфон через AltStore без платного Apple Developer.

## Что тебе нужно

- Аккаунт на GitHub (бесплатно)
- AltStore уже установлен на айфоне ✓
- AltServer запущен на маке ✓

## Шаги

### 1. Создай репо на GitHub

1. Зайди на github.com → New repository
2. Назови `PhotoSwipe`, можно публичный (тогда macOS-минуты бесплатные)
3. **Не** добавляй README/gitignore при создании — у нас уже свой
4. Создай

### 2. Залей код

Самый простой способ — через веб-интерфейс GitHub:

1. На странице нового пустого репо нажми **"uploading an existing file"**
2. Перетащи **всё содержимое** этой папки целиком (включая скрытые `.github`)
3. Если `.github` не видна: в Finder нажми `Cmd+Shift+.` чтобы показать скрытые папки
4. Commit changes

ИЛИ через терминал:
```bash
cd /путь/к/PhotoSwipe
git init
git add .
git commit -m "Initial"
git branch -M main
git remote add origin https://github.com/ТВОЙ_USERNAME/PhotoSwipe.git
git push -u origin main
```

### 3. Дождись сборки

1. На странице репо вкладка **Actions**
2. Увидишь запущенный workflow "Build iOS App"
3. Жди 3-5 минут пока стрелочка не станет зелёной галочкой
4. Кликни на завершённый запуск
5. Внизу страницы блок **Artifacts** → скачай `PhotoSwipe-ipa`
6. Это zip — распакуй, внутри `PhotoSwipe.ipa`

### 4. Установи через AltStore

1. Кинь `PhotoSwipe.ipa` на айфон любым способом:
   - AirDrop с мака
   - Через iCloud Drive
   - Через любое облако (Telegram Saved Messages работает)
2. На айфоне открой Files → найди `.ipa`
3. Долгий тап → Share → "Open in AltStore" (или просто открой через AltStore вручную → плюсик → выбери файл)
4. AltStore попросит Apple ID если ещё не вводил — введи
5. Ждёт 30-60 секунд, ставит
6. **Настройки → Основные → VPN и управление устройством → твой Apple ID → Доверять**
7. **Настройки → Конфиденциальность → Режим разработчика → Вкл** (если ещё не включал)
8. Открой PhotoSwipe с рабочего стола

### 5. При первом запуске

- Приложение спросит доступ к фото → **Разрешить → Все фото**
- Покажет первое фото из галереи
- Свайп влево → фото улетает в корзину (счётчик 🗑 сверху увеличивается, никаких диалогов)
- Свайп вправо → пропустить, следующее фото
- Когда накопил пачку — жми кнопку **🗑 N** сверху → одно системное подтверждение → вся пачка уходит в "Недавно удалённые"
- Если случайно свайпнул не в ту сторону — кнопка **↩** сверху отменяет последнее действие

## Что важно знать

- **Безопасность фоток:** удалённые фото попадают в альбом "Недавно удалённые" в Фото и хранятся там 30 дней — если ошибся, можно восстановить
- **Одно подтверждение на пачку:** iOS требует системный диалог при удалении из стороннего приложения, но он один на всю пачку, а не на каждое фото — поэтому копим и удаляем разом
- **7 дней:** приложение перестанет запускаться через 7 дней — переустанови через AltStore (он умеет это делать в один тап)
- **3 приложения:** на бесплатном Apple ID лимит 3 приложения одновременно; одно уже занято AltStore

## Если что-то пошло не так

**Workflow упал красным крестом:**
- Открой failed run → найди красный шаг → посмотри лог
- Скинь мне ошибку — починим

**AltStore говорит "Could not install":**
- Проверь что AltServer запущен на маке
- Проверь что айфон и мак в одной Wi-Fi сети
- Перезапусти AltServer

**Приложение упало при запуске:**
- Проверь iOS версию — нужна iOS 16+
- Скинь скриншот ошибки
